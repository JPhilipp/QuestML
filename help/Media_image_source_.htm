source Media image attribute
The source of an image, relative to the top QML folder. Typically, located in the "media" subfolder. You can use GIF, Animated GIF, JPEG and PNG images.

Paint Shop Pro 4 is a good program to create, edit and browse pictures. You can get it at Shareware.com (search for "Paint Shop Pro").
Sample:
<image source="media/treasure_chest.gif" />
<text>You open the treasure chest.</text>
