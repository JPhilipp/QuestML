Imagemap,usemap,graphics

Name Topic Used in Type
image Media station element
Includes an image to be shown with the station.

Paint Shop Pro 4 is a good program to create, edit and browse pictures. You can get it at Shareware.com (search for "Paint Shop Pro").

Images can be included multiple times. You can put them right in the station, or inside text content, or inside choices. Images can be positioned on the page by using the "class" attribute.
Sample:
<image source="media/treasure_chest.gif" />
<text>You open the treasure chest.</text>
[...]

Attributes: source | text | default | class
